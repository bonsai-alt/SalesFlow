# SalesFlow

Business sales tracker app — three interfaces, one database (Supabase).

## Apps
- `/manager` — Sales Manager app (mobile)
- `/owner` — Owner dashboard (desktop)
- `/combined` — Combined app with both roles

## Deployment
Each folder is a separate Netlify site pointing to its folder as publish directory.
Updates: edit files here → commit → Netlify auto-deploys in ~30 seconds.
